Solves the 0/1 Knapsack problem using dynamic programming. 

This function uses a bottom-\/up dynamic programming approach to find the most profitable subset of pallets that can be carried by the truck without exceeding its weight capacity. It builds a DP table to compute the maximum profit for each subproblem and then backtracks to determine the items that were included in the optimal solution.

\hline

\begin{DoxyNote}{Note}
The Dynamic Programming approach is pseudo-\/polynomial, due to the fact that the bigger the input the worse the running time is, because of the table. 

Time complexity\+: O(n $\ast$ W), where {\ttfamily n} is the number of pallets and {\ttfamily W} is the truck capacity. 

Space complexity\+: O(n $\ast$ W) due to the DP table. 
\end{DoxyNote}
