Solves the 0/1 Knapsack problem using a greedy approach. 

The algorithm selects pallets based on their profit-\/to-\/weight ratio, prioritizing items that offer the highest value per unit weight. This provides a suboptimal but efficient solution.


\begin{DoxyParams}{Parameters}

\hline
{\em capacity} & Maximum weight capacity of the truck. \\
\hline
\end{DoxyParams}
