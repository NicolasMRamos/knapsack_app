Solves the 0/1 Knapsack problem using a backtracking approach.

The algorithm tries various combinations (as a decision tree), pruning impossible branches and choosing the best.

Parameters
palletsReference to a vector of Pallet objects (defined in structs.h). Each pallet must have id, weight, and profit fields.
capacityMaximum weight capacity of the truck.
Returns
A std::pair where:
  • first is the total profit of selected pallets.
  • second is a vector of IDs of the selected pallets.
Note
Time complexity: O(2^n), in the worst case, tries all combinations.
Space complexity: O(n^2) due to recursion depth and copying of state vectors.
See also
backtracking.cpp for the implementation.
